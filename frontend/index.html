iCalCord

iCalCord is a Discord bot that generates iCalendar (.ics) feeds for Discord server events. It allows users to subscribe to their server's event calendar using any calendar application that supports the iCalendar format.

Use Guild ID as the last component in url, e.g.: /feed/1193841000108531764.ics

If you use Google Calendar, follow this guide to subscribe to the feed, then open Sync Settings and check the box for the calendar you just subscribed to, to make it propagate to non-Google mobile apps. If you're using Google Calendar app on Android, you need to open Settings, select the newly added feed on the list and enable "Sync".

If the server is not lurkable, you have two options:

  • Make the server lurkable
  • invite the bot to the server
